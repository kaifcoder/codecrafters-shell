üöÄ Getting Started

Prerequisites

  • Python 3.6+
  • uv package manager (recommended)
  • Unix-like environment (Linux, macOS, WSL)

Installation

# Clone the repository
$ git clone https://github.com/kaifcoder/codecrafters-shell.git
$ cd codecrafters-shell

# Install uv
$ curl -LsSf https://astral.sh/uv/install.sh | sh

# Run the shell
$ ./your_program.sh

üéØ Technical Highlights

Cross-Platform Compatibility

Detects GNU readline vs libedit (macOS), handles different key bindings, and uses OS-agnostic path handling.

Smart History Management

Tracks last write position per file for -a flag, prevents duplicate appends, and filters empty commands.

Robust Pipeline Execution

Handles builtin-to-external and external-to-builtin pipes with string buffering and proper process cleanup.
